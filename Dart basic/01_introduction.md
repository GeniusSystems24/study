# 1. مقدمة إلى Dart

## ما هي لغة Dart؟

Dart هي لغة برمجة حديثة تم تطويرها بواسطة Google في عام 2011. تم تصميمها لتكون لغة قوية ومرنة لبناء التطبيقات عبر منصات متعددة.

## لماذا Dart؟

### المميزات الرئيسية

- **سرعة التطوير**: دعم Hot Reload للتطوير السريع
- **أداء عالي**: تُترجم إلى كود native للأداء الأمثل
- **متعددة المنصات**: تطبيق واحد يعمل على Android, iOS, Web, Desktop
- **سهولة التعلم**: صياغة واضحة ومألوفة للمبرمجين
- **أدوات قوية**: نظام بيئي غني بالحزم والمكتبات

## استخدامات Dart

### 1. تطوير تطبيقات الموبايل (Flutter)
الاستخدام الأساسي والأكثر شهرة لـ Dart هو بناء تطبيقات Flutter.

### 2. تطبيقات الويب
يمكن تحويل كود Dart إلى JavaScript لتشغيله في المتصفحات.

### 3. تطبيقات سطح المكتب
بناء تطبيقات Desktop لـ Windows, macOS, Linux.

### 4. البرمجة من جانب الخادم (Server-side)
استخدام Dart لبناء APIs و backend services.

## تاريخ Dart

- **2011**: الإعلان عن Dart في مؤتمر GOTO
- **2013**: إصدار Dart 1.0
- **2018**: إطلاق Dart 2.0 مع نظام الأنواع القوي
- **2021**: Dart 2.12 مع Null Safety
- **2023-2025**: تحديثات مستمرة لتحسين الأداء والميزات

## العلاقة مع Flutter

Flutter هو إطار عمل UI تم بناؤه باستخدام Dart. عند تعلم Dart، فأنت تبني الأساس القوي لتطوير تطبيقات Flutter.
