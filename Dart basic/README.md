# خطة تعلم أساسيات Dart

## الفهرس

### [1. مقدمة إلى Dart](01_introduction.md)

**نبذة:** التعرف على لغة Dart، تاريخها، ولماذا تم تطويرها من قبل Google. فهم استخداماتها الأساسية خاصة في تطوير تطبيقات Flutter.

### [2. إعداد بيئة التطوير](02_setup.md)

**نبذة:** كيفية تثبيت Dart SDK، إعداد محرر النصوص (VS Code أو Android Studio)، وكتابة أول برنامج "Hello World".

### [3. المتغيرات وأنواع البيانات](03_variables.md)

**نبذة:** فهم المتغيرات (var, final, const)، أنواع البيانات الأساسية (int, double, String, bool)، والتحويل بين الأنواع.

### [4. العمليات والمعاملات (Operators)](04_operators.md)

**نبذة:** دراسة العمليات الحسابية، المنطقية، المقارنة، والعمليات الخاصة مثل ?? و ??= و cascade operator (..).

### [5. التحكم في التدفق (Control Flow)](05_control_flow.md)

**نبذة:** جمل الشرط (if, else, switch)، الحلقات التكرارية (for, while, do-while)، وعبارات التحكم (break, continue).

### [6. الدوال (Functions)](06_functions.md)

**نبذة:** كيفية إنشاء واستدعاء الدوال، المعاملات الإلزامية والاختيارية، Named Parameters، Arrow Functions، والدوال المجهولة.

### [7. المجموعات (Collections)](07_collections.md)

**نبذة:** التعامل مع القوائم (Lists)، المجموعات (Sets)، والخرائط (Maps)، والعمليات الشائعة عليها.

### [8. البرمجة الكائنية - الجزء الأول](08_oop_part1.md)

**نبذة:** إنشاء الأصناف (Classes)، المنشئات (Constructors)، الخصائص (Properties)، والطرق (Methods).

### [9. البرمجة الكائنية - الجزء الثاني](09_oop_part2.md)

**نبذة:** الوراثة (Inheritance)، التعددية الشكلية (Polymorphism)، الأصناف المجردة (Abstract Classes)، والواجهات (Interfaces).

### [10. Null Safety](10_null_safety.md)

**نبذة:** فهم مفهوم الأمان من القيم الفارغة، الفرق بين nullable و non-nullable types، واستخدام المعاملات الخاصة (!, ?., ??).

### [11. معالجة الأخطاء (Exception Handling)](11_exceptions.md)

**نبذة:** كيفية التعامل مع الأخطاء باستخدام try-catch-finally، إنشاء استثناءات مخصصة، وأفضل الممارسات.

### [12. البرمجة غير المتزامنة (Asynchronous Programming)](12_async.md)

**نبذة:** فهم Future و async/await، التعامل مع Streams، ومتى تستخدم البرمجة غير المتزامنة.

### [13. Generics (البرمجة العامة)](13_generics.md)

**نبذة:** استخدام الأنواع العامة لإنشاء كود قابل لإعادة الاستخدام، Generic Classes، وGeneric Methods.

### [14. المكتبات والحزم (Libraries & Packages)](14_libraries.md)

**نبذة:** كيفية استيراد المكتبات، استخدام pub.dev، إضافة التبعيات في pubspec.yaml، وإنشاء مكتباتك الخاصة.

### [15. مفاهيم متقدمة](15_advanced.md)

**نبذة:** Mixins، Extension Methods، Typedef، Enums، والتعبيرات المنتظمة (Regular Expressions).

### [16. أفضل الممارسات والأنماط](16_best_practices.md)

**نبذة:** اتفاقيات التسمية، تنظيم الكود، التعليقات والتوثيق، وأنماط البرمجة الفعالة في Dart.

### [17. التطبيقات العملية](17_projects.md)

**نبذة:** مشاريع عملية صغيرة لتطبيق ما تم تعلمه، مثل: آلة حاسبة، نظام إدارة المهام، أو تطبيق بسيط للتواصل مع API.
